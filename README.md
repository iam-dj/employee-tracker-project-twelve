![GitHub license](https://img.shields.io/badge/licens-MIT-black.svg)

## Description

This app is a command line interface that allows users to manage employee information in a MySQL database. With this app, you can view, add, and update employee information, such as their roles, departments, and managers. It also allows you to view all employees or filter by role or department. Overall, it's a convenient tool for managing employee data in a database.

## Installation

Before you start using the app, make sure to load the integrated terminal and type "npm i" to install all the necessary dependencies, including inquirer, express, and mysql2 packages. 

Once the installation is complete, make sure to load the schema in the SQL command line so that the tables are created. Then, load the seeds file so that there's some information in there for you to work with.

Now, start the application by typing "node index.js" in the command line, and follow the prompts!
